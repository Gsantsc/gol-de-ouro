# Guia de Instalacao PWA - Gol de Ouro

## Android

1. Abra o link do Gol de Ouro no Chrome.
2. Toque em `Instalar app` quando o banner aparecer.
3. Confirme a instalacao.
4. Abra o Gol de Ouro pela tela inicial.

## iPhone

1. Abra o link no Safari.
2. Toque no botao Compartilhar.
3. Toque em `Adicionar a Tela de Inicio`.
4. Confirme o nome `Gol de Ouro`.
5. Abra pela tela inicial.

## Desktop

1. Abra o link no Chrome ou Edge.
2. Clique no icone de instalar na barra de endereco, ou use o banner discreto do app.
3. Confirme a instalacao.

## Mensagem para enviar aos usuarios

```text

Abra o link abaixo no celular:

LINK_DO_APP

Android: toque em Instalar app.
iPhone: abra no Safari, toque em Compartilhar e depois em Adicionar a Tela de Inicio.
```
