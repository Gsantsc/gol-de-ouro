# Component Inventory

## Web Usuário

- `Shell`: layout base.
- `AuthPanel`: login/cadastro.
- `UserHeader`: resumo do usuário.
- `HomePanel`: próximos jogos, ligas e notificações.
- `GamesPanel`: lista de partidas.
- `MatchCard`: card de partida web.
- `PredictionDialog`: modal de palpite.
- `RankingPanel`: ranking global/grupos.
- `ProfilePanel`: perfil e estatísticas.
- `UserGroupsPanel`: ligas e convites.

## Admin

- `Shell`: layout base admin.
- `LoginCard`: autenticação admin.
- `Dashboard`: KPIs e saúde operacional.
- `UsersPanel`: busca, filtros, paginação e ações.
- `MatchesPanel`: gestão de partidas e status.
- `TournamentsPanel`: campeonatos.
- `GroupsPanel`: grupos.
- `CompetitionsPanel`: competições.
- `RankingPanel`: ranking admin.
- `ErrorBanner`: feedback de erro.

## Mobile

- `AppViewport`: frame com bottom navigation.
- `Screen` e `ScreenScroll`: layouts base.
- `Card`: superfície padrão.
- `AppButton` e `IconButton`: ações.
- `Field`: input padrão.
- `MetricTile`: KPI compacto.
- `EmptyState`, `ErrorState`, `ToastBanner`, `Skeleton`: estados de sistema.
- `BottomTabs`: navegação principal.
- `MatchCard`: partida compacta.
- Telas: Home, Jogos, Palpites, Ranking, Perfil, Grupos, Competições, Detalhes e Auth.
