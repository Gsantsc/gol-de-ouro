# User Dashboard UX Map

## Home

Resumo estratégico: saudação/header, posição, pontos, próximo jogo, palpites pendentes, palpites feitos, ligas ativas, notificações e atalhos para Jogos/Palpites.

## Jogos

Lista completa de partidas. Deve permitir busca por time, filtro por status e filtro por data. Mostra apenas indicador discreto quando já existe palpite.

## Palpites

Único local onde o usuário vê o placar enviado, status, pontos, resultado final e data de envio.

## Ligas/Grupos

Comunidade do usuário. Lista ligas com participantes, posição, pontos e entrada para detalhe.

## Ranking

Classificação geral e por liga, sem misturar gestão de grupo.

## Perfil

Dados pessoais, estatísticas resumidas e saída da conta.
